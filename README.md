# Wishlist App (Client-side localStorage)

## Fitur
- Create: tambah barang wishlist
- Read: lihat semua barang
- Update: edit barang & tandai sudah terbeli
- Delete: hapus barang
- Data tersimpan di localStorage

## Teknologi
- Vanilla JS + HTML + CSS

### Cara Menjalankan Menggunakan Vanilla JS
1. Cukup Buka file `index.html` lalu klik kanan search "Open In Default Browser"
   atau gunakan Live Server (VSCode).
